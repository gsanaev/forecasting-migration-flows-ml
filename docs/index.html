🌍 Forecasting Migration Flows with Machine Learning

An open, reproducible machine learning pipeline for analyzing and forecasting international migration dynamics (1990–2030) across 160+ countries.

🎯 Project Overview

This project explores how economic, demographic, and development indicators shape migration flows globally. Using open datasets from the World Bank and UNDP, it builds an interpretable machine learning model that forecasts net migration rates and simulates “what-if” development and crisis scenarios.

📅 Recent Update (October 2025)

The pipeline now includes forecasting through 2030 under four socioeconomic scenarios:

  • 📈 Baseline — steady continuation of current trends
  • 🚀 High Growth — +2 pp GDP growth, +3 % HDI
  • ⚠️ Crisis — −2 pp GDP growth, −2 % HDI
  • 👶 Demographic Pressure — +5 % adolescent fertility

Each scenario includes 90 % prediction intervals and regional aggregation summaries. Results are stored in outputs/forecast_results_2024_2030.csv and visualized in the final notebook.

📘 Workflow Notebooks

  • 🧹 Data Preparation & Cleaning — Collect, merge, and clean World Bank and HDI data.
  • 🔍 Exploratory Data Analysis (EDA) — Explore distributions, correlations, and missingness patterns.
  • ⚙️ Feature Engineering & Modeling — Build interpretable and interaction-based predictors for modeling.
  • 🌳 Model Interpretation & Scenario Analysis — Train Random Forest models and interpret them using SHAP values.
  • 📈 Forecasting & Validation — Evaluate temporal performance and produce scenario-based forecasts through 2030.

💡 Key Insights

  • 📉 Most countries show stable migration rates near zero, with crisis periods driving extreme outflows.
  • 🌱 Fertility decline and HDI growth are the strongest predictors of sustained migration inflows.
  • 💼 Economic expansion amplifies inflows in developed regions, while demographic pressure drives outflows in low-income countries.
  • 🌎 Forecasts for 2024–2030 reveal regional divergence between economic and demographic migration drivers.
